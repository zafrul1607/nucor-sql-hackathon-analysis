# Nucor SQL Hackathon Questions

## Overview

This repository documents SQL work completed for a Nucor-hosted SQL hackathon using the UC_NBT_Hackathon database and Recy schema.

- Time limit: 1.5 hours
- Team size: 4 people
- Required tasks: 1–7
- Bonus tasks: 8–11
- Analysis year: 2025

## Required Tasks

### Task 1: Monthly Shipment KPI Baseline

Create a monthly shipment KPI table for 2025 summarizing performance by location and month.

### Task 2: Top 3 Locations by Total Margin

Identify the three most profitable locations based on total shipment margin in 2025.

### Task 3: Top Consumers by Margin

For each top location, identify the five consumers that contributed the most margin.

### Task 4: Margin Trend and Running YTD Margin

For the consumers selected in Task 3, show monthly margin trend and cumulative YTD margin.

### Task 4B: Consumer Concentration

Within the same top consumer set, calculate how concentrated margin is in the number one consumer.

### Task 5: Monthly Trip KPI Baseline

Create a monthly trip KPI table to measure operational performance by location and month.

### Task 6: Top Drivers by Trip Volume and On-Time Rate

For each top location, identify the five most active drivers and their on-time performance.

### Task 7: Executive Profitability and Service Scorecard

Combine shipment profitability and trip service into one location-level ranking scorecard.

## Bonus Tasks

### Task 8: Month-End Inventory Totals by Location

Use inventory snapshots to compute month-end inventory totals per location and month-over-month change.

### Task 9: Inventory Drilldown for Top Location

For the highest-profit location, identify the top 10 inventories by month-end ending amount and show month-over-month deltas.

### Task 10: Trips Entered but Not Completed

Identify location-month combinations where trips were entered but not completed.

### Task 11: Self-Directed Analysis

Create an additional analysis and include two to three SQL comment insights.
